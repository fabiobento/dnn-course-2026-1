
# Deep Learning(DL) com PyTorch

Essa seção está estruturada para desenvolver suas habilidades em DL do zero:

* **Fundamentos:** Começaremos entendendo o básico, os neurônios de uma rede neural e, a matemática por trás deles.  
* **Prática com PyTorch:** Vamos ter aulas práticas, criando **Tensores** e visualizando como os dados se movem através de uma rede.  
* **Aplicações Reais:** Exploraremos desde sistemas de **Visão Computacional** que rivalizam com a visão humana até processadores de **Linguagem Natural** que entendem as nuances da nossa fala.
